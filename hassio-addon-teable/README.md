# [Teable](https://teablr.io) Community Edition Add-on for Home Assistant

This is a Home Assistant add-on that integrates Teable Community Edition, a no-code database solution.

## Features
- Runs Teable Community Edition within Home Assistant.
- Accessible via port 3000.
- Requires a PostgreSQL database (e.g., use the MariaDB or PostgreSQL add-on).

## Installation
1. Add this repository to Home Assistant: `https://github.com/yourusername/ha-teable-addon`.
2. Install the "Teable Community Edition" add-on from the Add-on Store.
3. Configure the `DATABASE_URL` in the add-on options.
4. Start the add-on.

## Requirements
- A running PostgreSQL instance (e.g., via another add-on).
